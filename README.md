# Styleplorer (Flexplorer)

Кастомный файловый проводник для Windows, разработанный на WPF.
Проект предоставляет базовый функционал стандартного проводника Windows и дополнительные возможности настройки интерфейса.

## Технологии

* C#
* WPF
* .NET 7
* Windows API

## Основные возможности

* Просмотр содержимого директорий
* Копирование, перемещение и удаление файлов и папок
* Поиск файлов и папок
* Просмотр свойств файлов и папок

## Дополнительные функции

* Изменение фона (изображение или GIF)
* Настройка прозрачности фона
* Изменение цвета или иконок папок
* Настройка цвета выделения
* Настройка тем оформления
* Настройка горячих клавиш для быстрого доступа

## Нестандартные возможности

* Работа с двумя окнами проводника
* Поддержка вкладок
* Перетаскивание вкладок между окнами
* Настройка размеров окна по умолчанию
* Теги для файлов и папок

## Запуск проекта

1. Клонировать репозиторий

```
git clone https://github.com/USERNAME/styleplorer-wpf.git
```

2. Открыть решение в Visual Studio

3. Собрать проект

```
Build → Build Solution
```

4. Запустить приложение

```
F5
```

## Требования

* Windows 10 / 11
* .NET 7 SDK
* Visual Studio
